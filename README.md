# Дипломная работа
Разработка и исследование механизма гарантированной транзакционной передачи данных в микросервисной архитектуре. В работе реализуется прототип системы на основе паттерна Transactional Outbox, обеспечивающего надежную доставку событий между сервисами без использования распределённых транзакций. Проводится анализ производительности, устойчивости к сбоям и потребления ресурсов при высокой нагрузке. Микросервисы, БД и брокеры сообщений завертывались в Kubernetes, в соответствующих папках есть yaml конфигурации этих систем
